This minor project was initiated in response to an informal request from a laboratory technician. After receiving an alert about a freezer malfunction, they were concerned about the potential temperature rise of a viral sample. They needed an estimate of the sample's temperature for the following morning when they returned to the lab. The primary challenge was determining the most accurate predictive model rather than the actual coding. The project took approximately 5 hours in total, and the resulting approximation proved to be highly accurate.
